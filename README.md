# ED - Unidad 07 

ED-Unidad07CPZ - Repositorio correspondiente a la **Unidad 07 de Entornos de Desarrollo**.
Ejercicios de Diagramas de Comportamiento   


## 📚 Descripción

En esta unidad se trabajan conceptos relacionados con el **modelado de sistemas mediante UML**, incluyendo:

- Diagramas de casos de uso
- Diagramas de actividades
- Diagramas de secuencia
- Diagramas de estados

El objetivo es aprender a representar el comportamiento y la estructura de un sistema de software de forma visual y comprensible.

---

## 🧩 Contenido del repositorio

Este repositorio incluye diferentes actividades prácticas:

### 🔹 Actividad 1: Centro Deportivo
- Casos de uso
- Diagrama de actividades (reserva de pista)

### 🔹 Actividad 2: Máquina de Billetes
- Diagrama de actividades (compra de billete)
- Diagrama de estados (objeto Billete)

### 🔹 Actividad 3: Sistema de Alarma
- Diagrama de estados
- Casos de uso

---

## 🛠️ Tecnologías utilizadas

- **PlantUML** → para la generación de diagramas
- **Markdown (.md)** → documentación
- Herramientas compatibles:
  - Visual Studio Code (con extensión PlantUML)
  - NetBeans
  - PlantUML Online
  - Diagram → https://app.diagrams.net
---


## 🎯 Objetivos de aprendizaje

- Comprender los distintos tipos de diagramas UML
- Modelar procesos y sistemas reales
- Interpretar flujos de ejecución
- Representar estados y transiciones
